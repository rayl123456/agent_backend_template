# agent_backend_template

## 项目定位

这是 AI Agent 智能体就业导向学习计划第一阶段的工程模板项目。

## 第一阶段目标

- 搭建 FastAPI 后端工程结构
- 实现文件上传接口
- 接入 SQLite / PostgreSQL 数据库
- 实现 Vue3 前端上传页面
- 使用 Docker Compose 一键启动后端、前端和数据库
